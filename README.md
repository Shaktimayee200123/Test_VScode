# MODbusUARTconvertionIN8version

This project implements Modbus UART conversion for STM32F103C8 microcontroller.

## Features

- Modbus protocol handling
- UART communication
- CRC calculation for Modbus

## Project Structure

- `Core/`: Main application code
- `Drivers/`: STM32 HAL drivers
- `Debug/`: Build artifacts

## Building

Use STM32CubeIDE to build the project.

## Usage

Flash the binary to STM32F103C8 and connect via UART for Modbus communication.
